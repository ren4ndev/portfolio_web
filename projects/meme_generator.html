Projeto Meme Generator

O projeto Meme-Generator é sobre o desenvolvimento de uma página Web que permita o upload de uma imagem e a inserção de um texto sobre ela, estilizado de forma apropriada. Nesse artigo, você verá alguns conceitos que foram aplicados durante esse projeto.

Para se tornar um bom desenvolvedor Front-End, uma das primeiras coisas que se deve aprender é como estruturar corretamento uma página HTML. O desafio de construir uma página geradora de memes fez com que fosse possível aprender e reforçar conceitos básicos de HTML, CSS e JavaScript. A proposta do projeto era construir uma página em que o usuário pudesse gerar um meme para redes sociais. Vej alguns dos requisitos do projeto:

  • A página deveria permitir que o usuário selecionasse em seu próprio dispositivo uma imagem de fundo para o meme, e disponibilizar algumas imagens pré-definidas.
  • O usuário deveria ser capaz de escrever o texto a ser usado no meme e escolher um dos estilos pré-definidos de borda e cor.

Para conseguir fazer isso foram usadas algumas técnicas simples de manipulação de elementos HTML com JavaScript e atribuição de classes aos elementos para realizar a estilização do documento. Embora tenha sido um projeto relativamente simples, este contribuiu para a consolidar o conhecimento básico de HTML5, CSS3 e JavaScript adquirido até então.

Você pode conferir o código neste repositório do GitHub. O arquivo README.md explica com detalhes os requisitos do projeto. Sinta-se à vontade para revisar e comentar o código - críticas construtivas são sempre bem-vindas. E lembre-se de que esse projeto foi desenvolvido no início da minha jornada para me tornar um desenvolvedor Web. Provavelmente eu mesmo hoje já acharia maneiras melhores de construir esse código e evitaria algumas más práticas. Mas o objetivo aqui é realmente expor pra quem quiser ver a minha trajetória como desenvolvedor, e o meu crescimento ao longo do tempo.

Conclusão

De qualquer forma, este projeto com certeza contribuiu muito para o aprendizado de novos conceitos e técnicas relacionadas ao desenvolvimento Web. Técnicas que serão muito úteis ao desenvolver aplicações reais.
