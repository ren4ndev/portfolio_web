Projeto Pixels Art

O projeto Pixels Art foi desenvolvido com o intuito de praticar a manipulação de elementos no DOM de acordo com os eventos de click no navegador. Nesse artigo você vai ver o que foi desenvolvido e o que eu aprendi com esse projeto.

A manipulação de eventos no DOM parece ser o primeiro passo para começar a adicionar interação à um página Web. O projeto Pixels Art foi uma ótima experiência para aprender e praticar as técnicas relacionadas a isso. O objetivo do projeto era criar uma página Web em que o usuário fosse capaz de pintar algo pixel por pixel. É claro que não se tratavam de pixels físicos, mas sim uma grade de elementos HTML sem nenhum preenchimento que simulavam os pixels de uma tela. Veja alguns dos requisitos especificados para a realização do projeto:

  • Toda vez que a página fosse carregada, uma paleta de cores aleatórias deveria ser gerada para o usuário.
  • Ao clicar em uma das cores da paleta, o usuário poderia "pintar" os pixels da cor escolhida, clicando neles

Embora simples, esse projeto ajudou a explorar bem o método addEventListener() do JavaScript, e também a usar a linguagem para modificar os atributos de cada elemento do DOM, o que possibilitava a troca de cores dos "pixels". Outra boa prática que começou a ser utilizada aqui foi a utilização de classes nos elementos, o que facilitou não somente a modificação dos estilos destes, mas também a manipulação deles via JavaScript.

Você poderá ver o código desenvolvido neste repositório do GitHub. O arquivo README.md explica com detalhes os requisitos do projeto. Sinta-se à vontade para revisar e comentar o código - críticas construtivas são sempre bem-vindas. E lembre-se de que esse projeto foi desenvolvido no início da minha jornada para me tornar um desenvolvedor Web.  Mas o objetivo aqui é realmente expor pra quem quiser ver a minha trajetória como desenvolvedor, e o meu crescimento ao longo do tempo.

Conclusão

Aprender o método do JavaScript addEventListener já abriu um grande leque de opções para adicionar interação às páginas da Web. A manipulação dos elementos no DOM também foram praticadas durante a criação dessa página. Com certeza, o projeto Pixels Art contribuiu muito para o meu crescimento como estudante de desenvolvimento Front-End.
