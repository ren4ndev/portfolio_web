Projeto Playground Functions

Durante a carreira como desenvolvedor é extremamente importante saber quebrar grande problemas em problemas menores e utilizar a lógica de programação para resolvê-los. Esse projeto foi um projeto didático para a praticar lógica de programação e alguns conceitos de JavaScript.

O projeto Playground Funtcions foi um projeto que permitiu colocar em prática vários conceitos importantes na escrita da linguagem JavaScript. Antes de implementar o projeto, uma rotina de estudos foi realizada para aprender ou relembrar conceitos básicos da linguagem como trabalar com Arrays, objetos e funções, operações aritméticas e a sintaxe da linguagem em geral.

O projeto continha 13 funções a serem implementadas. Note que a implementação das funções foi feita da forma mais simples possível, sem usar conceitos mais avançados do JavaScript. Mas a ideia era essa mesmo. O objetivo do projeto era não somente resolver os problemas propostos, mas também praticar a lógica de programação. Em muitos dos casos poderiam ter sido usados funções de alta ordem e outros métodos e bibliotecas do JavaScript para resolver os problemas. Mas todos os problemas foram resolvidos usando usando o JavaScript puro com condicionais e iterações simples.

Você pode conferir o código neste repositório do GitHub. O arquivo README.md explica com detalhes os requisitos do projeto. Sinta-se à vontade para revisar e comentar o código - críticas construtivas são sempre bem-vindas. E lembre-se de que esse projeto foi desenvolvido no início da minha jornada para me tornar um desenvolvedor Web. Mas o objetivo aqui é realmente expor pra quem quiser ver a minha trajetória como desenvolvedor, e o meu crescimento ao longo do tempo.
